选择器样式

插入形式分类

1.内联式

将css代码卸载标签中：

<标签 style="属性:值;属性:值">……</标签>
<p style="color:red;">123</p>

2.嵌入式

在<head>……</head>之间插入
<style type="text/css">

选择符     声明
选择符 {  属性:值  ;属性:值}
p      { color:blue}
</style>

为了修改方便也可再最后一条后加上分号
/*这是注释 和html的注释不同*/

3.外部式

将css代码写在一个单独的外部文件
以.css为扩展名，
在<head>内
用<link>标签
将css样式文件链接到html文件内：

<link href="abc.css" rel="stylesheet" type="text/css" />

三种方式的优先级

相同 权值 条件下

内联式>嵌入式>外部式

嵌入式>外部式有一个前提：嵌入式css样式的位置一定在外部式的后面。

也就是<style>在<link>后面

总的来说就是就近原则

其他形式

类/ID 样式

.类选择器名称{css样式代码;}

例如：

<span class="ZiMu">abc</span>	
.ZiMu{color:red;}

类选择器前面用(.)而ID选择器前面用(#)

类用class声明，ID用ID声明

ID选择器只能用一次，类选择器可用多次

……
.a{color:red;}
.b{font-size:20px;}
……
<span class="a b">123</span>

子选择器

用>符号指定标签元素的第一个子元素

.a>span{border:1px}

即a类所在标签下第一个<span>标签的样式


包含(后代)选择器

使用空格指定标签元素下的后辈元素

.a span{color:red;}

即a类所在标签内的所有标签的样式


通用选择器

用(*)指定html中所有标签

* {color:red;}

伪类选择符

其实和类(class)没有任何关系

是给不存在的标签（标签的某种状态）设置样式

例如给鼠标滑过的状态设置样式

a:haver{color:red}

给a标签（超链接）设置鼠标滑过显示为红色

兼容性很差 所以只给出常用的haver属性和<a>标签的组合


用(,)指定多个标签元素设置样式

.a,#b span{color:red;}
相当于
.a{color:red;}
#b span{color:red;}

继承

父元素种的某些样式可继承给后代元素

有的样式具有继承性

……
p{color:red;}
……
< p > abc < span > def < /span > < /p > 
……
< p > 标签下的 < span >标签也是被继承了红字样式 
而例如
p {border:1px;}就无法被继承

兼容性

权值

权值越高&nbps;优先级越高

类型 继承 标签 类 ID
权值 0.1 001 010 100

权值计算

#a .b p{}
权值：100+10+1=111

层叠

多个样式 相同权重时

根据先后顺序来决定：

p{color:red;}
p{color:green;}
< p > abc < /p >
其中abc是绿色字体
其实和靠近原则一样

重要性

用!important来将优先级提高到最高

p{color:red!important;}

注意：浏览器默认样式<网页制作者样式<用户自己设置的样式

CSS布局

标签元素分类

块状元素&nbps;内联元素（行内元素）&nbps;内联块状元素

常用块状元素：

<div>、<p>、<h1>...<h6>、<ol>、<ul>、<dl>、<table>、<address>、<blockquote> 、<form>

常用内联元素：

<a>、<span>、<br>、<i>、<em>、<strong>、<label>、<q>、<var>、<cite>、<code>

常用内联块状元素：

<img>、<input>


块状元素

特点：

每个元素从新的一行开始，且其后的元素也另起一行

高、宽、行高、顶/底边距可设置

默认时是父容器的100%


内联元素

特点：

和其他元素在一行上

高度、宽度、顶部和底部边距不可设置

宽度=包含的文字或图片的宽度，不可改变


内联块状元素

特点：

和其他元素在一行上

元素高度、宽度、行高、顶/底边距都可设置


盒子模型

padding=盒子与盒中内容的距离

margin=盒子于盒子的距离

border=盒子边框

内填充、外边距、边框都有四个方向

top/bottom/left/right

border:2px solid red;
}

border-style:solid;
border-color:red;
}

dashed虚线 dotted点线 solid实线	

border-width还包括
thin midium thick

大小

一个元素的“占地面积”由长宽决定

weight和height只是最内层内容的长和宽

整个元素的宽度 =
左间距margin-left +
左边框粗细border-left +
左填充padding-left +
内容宽度weight +右填充+右边框粗细+右间距

填充

顺时针 上 右 下 左
也可

	padding-right:20px;

边界

margin：元素与其他元素只见的间隔

同理，只是margin在边框外，padding在边框内


CSS布局模型

分为三种布局模型：

流动模型Flow（默认）、浮动模型Float、层模型Layer


流动模型Flow（默认）

块状元素都会在所处的包含元素内自上而下按顺序垂直延伸分布

因为在默认状态下，块状元素的宽度都为100%。实际上，块状元素都会以行的形式占据位置。

举例：div标签中即便设置了较低宽度，也会占据一行。

在流动模型下，内联元素都会在所处的包含元素内从左到右水平分布显示。


浮动模型Float

可使用float:left;将元素设置为居左


层模型

类似ps里的图层，有三种形式：

绝对定位position:absolute、相对定位position:relative、固定定位position:fixed

绝对定位

position:absolute将元素使用left、right、top、bottom等相对其最接近的一个具有定位属性的父包含块进行绝对定位

即 相对于父元素定位


相对定位

使用position:relative,通过left/right/top/bottom确定在正常文档流中的偏移位置

首先按static(float)方式生成一个元素然后相对于以前的位置进行移动，移动前的位置保留


固定定位
