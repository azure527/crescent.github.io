插入形式分类

内联式

将css代码卸载标签中：

<标签 style="属性:值;属性:值">……</标签>
<p style="color:red;">123</p>

嵌入式

在<head>……</head>之间插入
<style type="text/css">

选择符     声明
选择符 {  属性:值  ;属性:值}
p      { color:blue}
</style>

为了修改方便也可再最后一条后加上分号
/*这是注释 和html的注释不同*/

外部式

将css代码写在一个单独的外部文件
以.css为扩展名，
在<head>内
用<link>标签
将css样式文件链接到html文件内：

<link href="abc.css" rel="stylesheet" type="text/css" />

三种方式的优先级

相同 权值 条件下

内联式>嵌入式>外部式

嵌入式>外部式有一个前提：嵌入式css样式的位置一定在外部式的后面。

也就是<style>在<link>后面

总的来说就是就近原则

类/ID 样式

.类选择器名称{css样式代码;}

例如：

<span class="ZiMu">abc</span>	
.ZiMu{color:red;}

类选择器前面用(.)而ID选择器前面用(#)

类用class声明，ID用ID声明

ID选择器只能用一次，类选择器可用多次

……
.a{color:red;}
.b{font-size:20px;}
……
<span class="a b">123</span>

子选择器

用>符号指定标签元素的第一个子元素

.a>span{border:1px}

即a类所在标签下第一个<span>标签的样式


包含(后代)选择器

使用空格指定标签元素下的后辈元素

.a span{color:red;}

即a类所在标签内的所有标签的样式

通用选择器

用(*)指定html中所有标签

* {color:red;}

伪类选择符

其实和类(class)没有任何关系

是给不存在的标签（标签的某种状态）设置样式

例如给鼠标滑过的状态设置样式

a:haver{color:red}

给a标签（超链接）设置鼠标滑过显示为红色

兼容性很差 所以只给出常用的haver属性和<a>标签的组合


用(,)指定多个标签元素设置样式

.a,#b span{color:red;}
相当于
.a{color:red;}
#b span{color:red;}

继承

有的样式具有继承性

……
p{color:red;}
……
< p > abc < span > def < /span > < /p > 
……
< p > 标签下的 < span >标签也是被继承了红字样式 
而例如
p {border:1px;}就无法被继承

兼容性

权值

权值越高&nbps;优先级越高

类型 继承 标签 类 ID
权值 0.1 001 010 100

权值计算

#a .b p{}
权值：100+10+1=111

层叠

多个样式 相同权重时

根据先后顺序来决定：

p{color:red;}
p{color:green;}
< p > abc < /p >
其中abc是绿色字体
其实和靠近原则一样

重要性

用!important来将优先级提高到最高

p{color:red!important;}

注意：浏览器默认样式<网页制作者样式<用户自己设置的样式

文字排版

样式

属性：

font-family:宋体		设置字体为宋体
font-size = 12px;		字号=12
color = red;			字色=红色
line-height = 1.6em;		行间距1.6em (1em=16px=1个文字大小)
border:1px solid red;		1px粗的红色边框 边框颜色为红色
font-weight:bold;		粗体字
font-style:italic;		斜体字
text-decoration:underline;	下划线
text-decoration:line-through;	删除线
text-indent:2em;		缩进2个文字大小
line-height:2em;		行间距（行高）2个文字大小
letter-spacing:20px;		字间距20px宽
word-spacing:20px;		英文单词间距20px宽
text-align:center;		块状元素（标签）内容居中
